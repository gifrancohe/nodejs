# nodejs

Introducción a NodeJS mediante el desarrollo de un proyecto ágil

Primera entrega
* desde la consola y en el directorio raiz del proyecto ejecutar el comando npm -i, esto con el fin de instalar las dependencias necesarias.
* Para ejecutar el proyecto se debe ingresar el comando node index, esto listará la oferta de cursos disponibles.
* Para realizar la inscripición se debe hacer mediante el comando node index inscribir.
* Para ver los parametros requeridos se puede utilizar el comando node index inscribir --help
* Un ejemplo de inscripción a uno de los cursos sería el siguiente:
node index inscribir -i=2 -n=Gio -d=123
* Luego de realizar un proceso de inscripción en alguno de los cursos, se creará el archivo txt y adicional se mostrara en el navegador que tenga predeterminado los resultados de la inscripción anterior.
* Si desea parar la ejecución del script, utilize el comando Ctrl + C
